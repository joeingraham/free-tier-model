# Free Tier Model — Copilot

Interactive financial model for Copilot Free → paid conversion economics.

**[Open the Dashboard →](https://joeingraham.github.io/free-tier-model/)**

## What It Models

Free users have an ongoing COGS cost. Each month they either convert to paid, churn, or stay free. The model simulates the full cohort dynamics and computes per-user expected LTV, CAC, and payback.
